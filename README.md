# JavaScript Todo App

A simple and responsive Todo App built using HTML, CSS, and Vanilla JavaScript.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as completed
- ✅ Delete tasks
- ✅ Add tasks using the Enter key
- ✅ Input validation
- ✅ Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Use

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Start adding your daily tasks.

## Future Improvements

- Local Storage
- Edit Task
- Dark Mode
- Task Categories
